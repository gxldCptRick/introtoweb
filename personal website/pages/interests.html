Favorites

I have many favorite things such as video games and movies but my most favorite thing is computers. I know how bad that must sound me talking about computers all the time. I digress but still computers are one of the most intergral things in our modern lives. It helps us from waking us up to determining the exact number of pi. Computers will always be neccesary and that is why I find computers to be my favorite thing. Because it is so useful from mundane day to day task to crucial life support systems on space crafts. This has lead me to keep researching computer programming so that I can understand just how everything fits together. In my Journey I have found many different things that I would have never realized. Like how every computer is running an Operating System program that is just running an infinte loop iterating over certain basic comands and only exits when it hits a break point. This enourmous revelation has lead to see how I can use loops to my advantage and how all my predecessors have used them to make all of our lives easier. It also showed me why there are such things as computer genereted errors on constantly running systems. It is because the machine is still running on the total access of memory and if it doesn't manage its memory well. Or not even managing it well but it needs to keep a constant growth in memory and that eventual takes up all of it.
